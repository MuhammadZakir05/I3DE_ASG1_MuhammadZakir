# I3DE_ASG1_MuhammadZakir

## Brief
- You play as the facility worker who somehow wakes up alone in the building. Everything is locked and you are trapped, the only possible way to escape is to restore the power to the facility. In order to do so, you must scavenge around the building for 5 power fuses and repair the main generator.

## Controls
- WASD: Move
- Mouse: Look around
- E: Interact / open doors / collect items / repair generator

## Game Objective
Escape the science facility.

To win:
1. Collect 5 fuses.
2. Find the keycard.
3. Repair the generator.
4. Unlock the Experiment Room / Area 2 gate.
5. Reach the exit (upon going through area 2).

## Puzzle Answers / Game Hack
- Required fuses: 5
- Keycard is needed for the Area 2 gate.
- Generator must be repaired before Area 2 can be unlocked.
- Password door code: 6920 (Warehouse Door password)

## Platform / Hardware
- Platform: Windows PC

- Recommended: Keyboard and mouse
- Display: 1920x1080 recommended
